## Setup
Clone this repo to your desktop and run `npm install` to install all the dependencies.
## Install

```sh
npm install
```

## Usage

Development

```sh
npm run dev
```
Production Build

```sh
npm run build
```

## Run tests

```sh
npm run test
```
